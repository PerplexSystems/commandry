# Examples

This directory contains example programs demonstrating commandry's features.

- [`hello.sml`](./hello.sml) - Minimal example with a single positional argument
- [`add.sml`](./add.sml) - Two positional arguments for adding numbers
- [`simple.sml`](./simple.sml) - Flags, options, and positional arguments combined
- [`git-like.sml`](./git-like.sml) - Multiple flags with a variadic positional list
- [`subcommands.sml`](./subcommands.sml) - Nested subcommands with distinct parsers
